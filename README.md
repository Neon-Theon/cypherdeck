# CypherDeck – Gemini AI Cyberpunk Typing RPG
<img width="1502" height="998" alt="Screenshot 2026-03-09 012228" src="https://github.com/user-attachments/assets/2f5c0047-fb2c-4055-ade8-1efed93fae4f" />

**Built entirely in Google AI Studio with Gemini**  
A dynamic touch-typing role-playing game in a cyberpunk hacking style. Improve your typing skills by completing 'hacking missions' generated by Gemini, earning XP, and leveling up your abilities in a neon-drenched digital underworld.

## ✨ Features
- Real-time AI-generated hacking missions with unique objectives and lore every playthrough
- Live typing challenges with speed (WPM) and accuracy scoring
- Character progression system — earn XP and unlock new abilities
- Full cyberpunk terminal interface with immersive effects
- Adaptive difficulty that scales with your performance

## 🚀 How to Run
1. `npm install`
2. Copy `.env.example` to `.env.local` and add your `GEMINI_API_KEY`
3. `npm run dev`
4. Open http://localhost:5173

## 🛠️ Tech Stack
- Google AI Studio + Gemini
- Vite + React + TypeScript
